# 🌦️ AI Weather Forecast App

> Real-time weather · AI assistant · Animated effects · Disaster alerts

## 🚀 Deploy in 3 steps

### Step 1 — Upload to GitHub
1. Go to github.com → New repository → name it `ai-weather-app`
2. Upload all these files (drag and drop)

### Step 2 — Deploy on Vercel
1. Go to vercel.com → Add New Project
2. Import your GitHub repo
3. Add environment variable:
   - Key: `VITE_ANTHROPIC_KEY`
   - Value: your key from console.anthropic.com
4. Click Deploy ✅

### Step 3 — Done!
Your app is live at `https://ai-weather-app-yourname.vercel.app`

## 💻 Run locally
```bash
npm install
npm run dev
```
Then open http://localhost:3000
