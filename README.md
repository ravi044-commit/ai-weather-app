# 🌦️ AI Weather Forecast

A premium weather application powered by Claude AI — featuring real-time 
weather data, stunning animated effects, and an intelligent assistant.

## ✨ Features
- 🌍 Search any city worldwide
- 🎨 Dynamic glassmorphism UI with time-based backgrounds
- 🌧️ Animated rain, snow, thunderstorm & fog effects
- 🤖 AI weather assistant (ask anything)
- 📅 7-day forecast with rain probability
- 📊 24-hour temperature & rain charts
- 🌿 Air Quality Index (AQI) monitor
- 🚨 Real-time earthquake & disaster alerts
- 🌕 Moon phase tracker
- 📳 Vibration alerts on mobile
- 🗣️ Japanese emergency voice alerts
- ⚡ Built with React, Vite & Recharts

## 🛠️ Tech Stack
React · Vite · Recharts · Lucide Icons · Claude AI · Web Search API
